# Uber-Power-BI-dashboard
Interactive Power BI dashboard analyzing Uber ride demand, revenue, vehicle performance, and ride completion trends.
<img width="1515" height="797" alt="image" src="https://github.com/user-attachments/assets/0df360e1-4302-4a23-9568-0759ce74165f" />
## Dashboard Insights
- Peak ride demand observed during evening time slots
- Auto vehicle type generated highest revenue
- Monthly ride completion trends were analyzed
- Ride demand patterns varied across weekdays and time slots

## Tools Used
- Power BI
- Power Query
- DAX
- Excel
